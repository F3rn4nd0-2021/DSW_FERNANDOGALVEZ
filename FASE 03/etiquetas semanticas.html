¿QUÉ SON LAS ETIQUETAS SEMÁNTICAS?

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores.

¿PARA QUE SIRVEN LAS ETIQUETAS SEMÁNTIAS ?

Las etiquetas nos permiten añadir información a la Web, como metadatos semánticos que puedan ser interpretados por cualquier sistema informático. Con ello se puede facilitar que cualquier máquina que lee un documento HTML y no está apreciando cómo se presenta en pantalla, sino simplemente su código fuente, pueda saber a qué parte típica pertenece cada pedazo de código, para entender mejor qué es lo que se está tratando en cada caso.

TIPOS DE ETIQUETAS SEMÁNTICAS

1- Elemento < section > HTML:

El elemento < section > define una sección en un documento.
Una sección es una agrupación temática de contenido, normalmente con un encabezado
EJEMPLO

2- Elemento < article > HTML

El < article > elemento especifica contenido autónomo e independiente.
Un artículo debe tener sentido por sí solo y debe ser posible distribuirlo independientemente del resto del sitio web.
EJEMPLO

3- Elemento < header > HTML

El < header > elemento representa un contenedor para contenido introductorio o un conjunto de enlaces de navegación. Un < header > elemento normalmente contiene:
  • uno o más elementos de encabezado (< h1 > - < h6 >)
  • logo o icono
  • información de autoría
    • EJEMPLO

    4- Elemento HTML < footer >

    El < footer > elemento define un pie de página para un documento o sección.
    Un < footer >elemento normalmente contiene:
  • información de autoría
  • informacion registrada
  • Información del contacto
  • mapa del sitio
  • volver a los enlaces superiores
  • documentos relacionados
    • EJEMPLO

    5- Elemento HTML < nav >

    El < nav> e lemento define un conjunto de enlaces de navegación.
    EJEMPLO

    Elemento HTML < aside >

    El < aside > elemento define algún contenido además del contenido en el que se coloca (como una barra lateral).
    El < aside > debe estar relacionado indirectamente con el contenido circundante.
    EJEMPLO

    Elementos HTML < figure> y < figcaption >

    La etiqueta< figure > especifica contenido autónomo, como ilustraciones, diagramas, fotos, listas de códigos, etc.
    La etiqueta < figcaption > define un título para un < figure > elemento. El < figcaption > elemento se puede colocar como el primero o el último hijo de un < figure > elemento.
    El elemento < img > define la imagen / ilustración real.
